- 此处会列出所有我在电脑中安装的软件，科普的同时做一个个人记录用，防止下次装系统的时候忘了自己需要什么软件，所有软件排名不分先后。
- 我个人使用的电脑为Windows系统，Mac/Linux用户可以考虑划走了。


## 装机必备类

### 火绒
- https://huorong.cn/
- 我个人不习惯用Windows Defender，基本上重装完系统后先安装火绒，这是一款很出名的国产杀毒软件，但和360之类的其他国产杀毒软件不同，这款软件非常的干净。类似的杀毒软件还推荐卡巴斯基，ESET等国外杀毒软件，但这些需要付费。

### Everything
- https://www.voidtools.com/zh-cn/downloads/
- 一款文件搜索工具，速度比用Windows自带的文件搜索快很多。顺带一提，其实Everything还有很多函数可以用来辅助搜索，简单的了解并加以熟练的应用一定能事半功倍。

### PotPlayer
- https://potplayer.tv/
- 非常好用的媒体文件播放软件，支持的解码格式以及功能很多。

### Eartrumpet
- https://eartrumpet.app/
- 可以每个应用独立调整音量。

### 7-zip
- https://7-zip.org/
- 非常轻量化的压缩软件，就是可能界面有点简陋。

### Google Chrome
- https://www.google.com/chrome/
- 有段时间想过换Edge，但感觉还是用不顺手，依旧用着Chrome。虽然不是很喜欢Chrome加了一大堆AI有关的东西。

### Iris (付费软件)
- https://iristech.co/zh-hans/
- 一款好用的防蓝光软件，我个人的眼睛是受不了长时间盯屏幕的，所以新装好系统后也会选择装一个Iris，比Windows自带的护眼强。

### Microsoft Office
- https://office.com/
- 虽然WPS也在迅速发展并且有了许多本地化的功能，但我依旧选择Microsoft Office。

## 编程类

### Git
- https://git-scm.com/
- 没有Git，可以说后面一切免谈。

### VSCode
- https://code.visualstudio.com/
- 曾经是是用jb家的东西的，但是感觉不够轻量化，逐步换到VSCode了。

### Fiddler
- https://www.telerik.com/download/fiddler
- 一个好用的抓包工具，而且使用门槛不高，小白也可以比较容易地上手。

### WireShark
- https://www.wireshark.org/
- 网络数据包分析软件，但感觉使用门槛比Fiddler略高，这两个软件的功能也仅仅是有交集，但大部分不相同。

### Github Desktop
- https://github.com/apps/desktop
- 如果你不是像我一样一开始就被带偏了开始用Github Desktop，那请你不要用他，没有任何好处。

### Python
**其他更多编程语言不再一一列出**
- https://www.python.org/
- 很长一段时间我都在使用Python 3.8，感觉新版用不惯，但由于兼容性问题也逐步替换了。

### Docker
- https://www.docker.com/
- 偶尔在测试时也需要在Windows中跑Docker容器，虽然我感觉很傻逼。

### Sublime Text
- https://www.sublimetext.com/
- 配置好之后一定是你的利器，我一般把它当加强版的笔记本用。

### Claude Code
- https://claude.com/product/claude-code
- Vibe Coding普及我感觉是迟早的事儿。

### phpStudy
- https://www.xp.cn/phpstudy/
- 依旧不推荐，但用了之后离不开了。

## 验机跑分类

### 图吧工具箱
- https://www.tbtool.cn/
- 这里头的东西挺全的了，下面介绍的软件大部分在该工具箱中都有。

### CPU-Z (在工具箱中)
- https://www.cpuid.com/softwares/cpu-z.html
- 可以查看CPU，内存，主板等的具体信息，可以进行简单的CPU跑分测试。

### FurMark (在工具箱中)
- https://geeks3d.com/furmark/
- 俗称甜甜圈，GPU烤机的时候用。

### ASSDBenchmark (在工具箱中)
- https://www.alex-is.de/PHP/fusion/infusions/downloads/downloads.php
- ~~写这篇文章的时候才发现自己从来没去过ASSSDBenchmark的官网，这也是网上找的，不确定是不是官网~~ 比较经典的硬盘测试工具，可以测试顺序读写，4k读写等的速度。

### CrystalDiskInfo (在工具箱中)
- https://crystalmark.info/en/software/crystaldiskinfo/
- 可以查看硬盘的详细信息，记得切换主题。

## 小工具类

### WizTree (在工具箱中)
- https://diskanalyzer.com/
- 可以直观的看到磁盘中每个文件占用的大小，删除无用东西的时候很管用。

### Dism++ (在工具箱中)
- https://www.chuyu.me/zh-Hans/
- 本身就是一个小工具箱，有很多有用的有关系统优化的工具。

### Wise Disk Cleaner
- https://www.wisecleaner.com/wise-disk-cleaner.html
- 垃圾清理软件，扫盘速度比较快，个人感觉还是很好用的。

### Geek Uninstaller
- https://geekuninstaller.com/
- 一个非常轻便的软件卸载工具，可以同时删除残留的文件和注册表等

### Bandicam (需购买)
- https://www.bandicam.cn/
- 好用的录屏软件，可以在网上找一下便携版。

### 搞机工具箱
- https://jamcz.com/gjgjx/
- 一个知名b站UP主开发的，算是比较好用的折腾安卓手机的工具箱。

### Cloudreve
- https://cloudreve.org/
- 一个云盘工具，我有时拿这个东西临时传文件用。

### FFmpeg
- https://www.ffmpeg.org/
- 就和你的电脑会有很多Chromium一样，你的电脑一定会有很多个FFmpeg，但我还是会单独安装一个并添加到PATH中。

### LX Music

### Apple Music (需订阅)
- https://music.apple.com/
- 应该是Apple支持平台最多的应用。Apple Music有最全的欧美乐和古典乐，音质也是独一档的，可惜日语歌稍微较少。使用也不想国内音乐平台那么多杂七杂八的东西，适合喜欢纯粹听歌的用户使用。

### Steam
- https://store.steampowered.com/
- 感谢家庭组让我嫖了个黑猴。

### Epic
- https://store.epicgames.com/
- 我的E宝！

### 原神
- https://www.yuanshen.com/
- 不说了，原神，启动！

### BetterGI
- https://www.bettergi.com/
- 一个辅助玩原神的工具，可以自动拾取自动刷本等。
